This is my first website created in terminal using vi commands

Linux Commands

CD = Change Directory
Mkdir NAME = Make Directory
Ls = list files
Vi FILENMAE = replace with the actual file mode

VI COMMANDS

i = insert mode
Escape = get out of insert mode
Shift key + colon = command prompt for vi
w = command to write
q= command to Quite
wq = write and Quit

Git COMMANDS

git init = Create the repository
git add filename = add the file(s) to git in preparation for a commit
git commit -m "message"= do the commit to save changes
git checkout branch name = make a new branch and checkout
git merge BRANCH NAME = merge the branch Note: you have to be on the branch that you want to merge to
